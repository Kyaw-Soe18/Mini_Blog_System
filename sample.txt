DB Connection
=============
. create database in phpmyadmin
. env (configur) connection 
. php artisan migrate

Table Create 
============
php artisan make:migration create_products_table

DB command 
==========
php artisan migrate             => upload all table from database / migration
php artisan migrate:rollback    => roll back all table from phpmyadmin
php artisan migrate:status      => check tables upload or not
php artisan migrate:refresh     => rollback and then migrate 
php artisan migrate:fresh       => rollback and then migrate

Making Model 
============
php artisan make:model User     => model create
php artisan make:model User -m  => model + table create
php artisan make:model User -mc => model + table + controller create
